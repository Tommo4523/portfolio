Rocket Lab: Neutron Thrust Module

Structural analysis, qualification testing and design release for Neutron thrust module structures and fluid system components.

Project Overview

During a three-month internship on Rocket Lab's Neutron program, I worked within the thrust module team, the primary structure that transfers thrust from the seven Archimedes engines into the vehicle. My work spanned structural analysis and design iteration, planning and executing qualification tests, and carrying components through the full engineering release process to manufacture.

Key Responsibilities

  • Structural analysis and iteration. Performed detailed FEA on thrust module components using Siemens NX Simcenter (NASTRAN), reporting findings and leading next-step discussions with the team. Iterated designs through multiple cycles to close out requirements within a fast-paced launch vehicle development schedule.
  • Sub-system qualification testing. Designed and executed validation tests for large Neutron sub-systems, working across the full test lifecycle including test plans, fixture design, instrumentation, real-time execution and post-test data reduction and analysis.
  • End-to-end design release. Carried components from concept through detailed 3D CAD, drawing packages to ASME Y14.5, PLM release in Teamcenter, and into manufacture. Owned the full lifecycle of assigned parts including fluids mounts that interface with the Archimedes engines.
